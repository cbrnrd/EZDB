# ezdb

Welcome to ezdb! ezdb is a small and simple NoSQL database. More languages are going to come in the future.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  ezdb:
    github: cbrnrd/ezdb
```

## Usage

```crystal
require "ezdb"
```

To start the daemon, build main.cr and run it.
