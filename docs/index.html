EZDB

An in-memory NoSQL database used for IPC.

FEATURES

Simple

ezdb is written in Crystal. The code is easy to read, and it is easy to set up

Lightweight

The binary for ezdb is incredibly small and uses minimal system resources

INSTALLATION

The best way to install ezdb is through git. The crystal compiler is required (for now).

GIT
$ curl https://git.io/vhIl7 | sh

IMPLEMENTATIONS

There are client programs in many different languages for interacting with ezdb:

  • Java - ezdb4j
  • Ruby - ruby-ezdb (Coming soon)
  • Python - PyEzdb (Coming soon)
  • Go - go-ezdb (Coming soon)
